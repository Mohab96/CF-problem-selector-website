This app assists CPers in the process of choosing problems while training
Getting started:
  • You will get problems with the number you entered.
  • The problems you will get will have ratings between the boundaries you entered.
  • The problems won't be solved by any handle in the handles field.
  • You can enter handles one-by-one, or multiple handles comma-separated in one shot.
  • The 'Save' button will save your preferences locally and will be loaded automatically when you open/reload the site.
  • The 'Reset' button will remove what is stored locally.
  • The problems you will get won't have any tag which is not within selected tags.
  • If the user selected number theory, math, and dp, then all these combinations are valid and can be seen in the outputted problems:
    • number theory
    • math
    • dp
    • dp, math
    • dp, number theory
    • math, number theory
    • math, number theory, dp
    but you wont see any of these:
    • binary search, math
    • combinatorics
    • etc.
  • If no tags are selected, then all tags will be included by default as if you have selected all of them.
  • Only codeforces rated problems will be included.
  • Any problem with no tags will not be included.
  • To remove a handle, Click on it
